# 🌐 Portafolio Web Profesional

![Versión](https://img.shields.io/badge/version-v0.2.0-blue)

Este proyecto consiste en un **sitio web de portafolio personal** desarrollado con **HTML5 y CSS3**, diseñado para mostrar información profesional, proyectos y habilidades de forma clara, moderna y responsive.

El objetivo del proyecto es crear una **estructura limpia, escalable y profesional** que pueda servir como base para un portafolio de desarrollador.

# 📌 Características

- Diseño moderno y limpio
- Estructura semántica con **HTML5**
- Diseño **responsive** adaptable a móvil y escritorio
- Layout basado en **Flexbox**
- Código organizado y fácil de mantener
- Paleta de colores profesional
